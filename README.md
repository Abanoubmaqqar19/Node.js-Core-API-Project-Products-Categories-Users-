# 📦 Node.js Core API Project

## Overview

This project is a **Node.js API built with core modules** (no Express) that supports CRUD operations for:

- Products
- Categories
- Users

  ---

# Concepts Learned:

- Node.js HTTP module
- Routing
- Handling request body streams
- JSON parsing and response
- CRUD logic
- Error handling
- CommonJS modules

---

# Testing with Postman

## Products
- `GET /api/products` → all products
- `POST /api/products` → add product
- `PUT /api/products/:id` → replace product
- `PATCH /api/products/price/:id` → update price
- `DELETE /api/products/:id` → delete product

## Categories
- `GET /api/categories` → all categories
- `POST /api/categories` → add category

## Users
- `POST /api/users` → add user
- `PATCH /api/users/:id` → update password
- `DELETE /api/users/:id` → delete user

**Note:** Always use `Content-Type: application/json` for POST/PATCH.
